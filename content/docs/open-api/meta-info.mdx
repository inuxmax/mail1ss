title: Meta Scraping API
description: Extract website meta information
---

##  Usage

API URL:
```bash
https://Mail1s.net/api/v1/scraping/meta?url=https://www.example.com&key=your_api_key
```

API Response:
```json
{
  title: "",
  description: "",
  image: "",
  icon: "",
  url: "",
  lang: "",
  author: "",
  timestamp: "",
  payload: "",
}
```
